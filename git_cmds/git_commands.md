# git commands
**list git branches**
```
git branches
```
**create a new branch**
```
git checkout -b <BRANCH_NAME>
```
**switch branch**
```
git checkout main
```
**delete branch locally**
```
git branch -d <BRANCH_NAME>
```
**get commit logs/ ids**
```
git log
```
**revert changes**
```
git revert <COMMIT_ID>
```
